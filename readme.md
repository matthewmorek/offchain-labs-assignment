# Offchain Labs Assignment

A Next.js application with three task pages demonstrating various frontend implementations.

## Getting Started

### Prerequisites

- Node.js 22+
- npm/pnpm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
npm start
```

## Pages

- `/task-1` - CMS Schema documentation
- `/task-2` - SVG Stripe Animation
- `/task-3` - Navigation Menu & Newsletter Signup

## Documentation

Task-specific documentation is available in the `docs/` directory.
